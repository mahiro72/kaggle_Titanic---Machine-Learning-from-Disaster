## 仮想環境作成について
- python -m venv myenv
myenvという名前の仮想環境を作成します

- cd OOO/OOO/
cd を使ってコマンドプロンプトでmyenvのディレクトリまで移動します

- myenv\Scripts\activate
仮想環境をアクティベートします

- pip install -r requirements.txt
注意 現在のディレクトリにrequirements.txtおいてください

以上で、仮想環境の作成完了です
